# Structure

- **common/**: Contains reusable classes and models shared between client and server (e.g., Patient, Service, Doctor).  
- **server/**: Handles server logic, data management, and connections from clients.  
- **client/**: Provides a GUI with forms for interacting with the application (adding/viewing patients, doctors, receipts, services).
